# Studi Kasus: PCA dan Autoencoder pada Data Berdimensi Tinggi

Bagian ini membandingkan kinerja reduksi linier (PCA) dan kompresi non-linier berbasis *autoencoder* pada dataset berdimensi tinggi seperti piksel citra. Kita mengevaluasi bagaimana karakteristik kedua pendekatan reduksi tersebut memengaruhi performa tugas klasifikasi lanjutan.

Pendekatan pertama menerapkan PCA, metode linier yang memproyeksikan data ke sumbu ortogonal baru untuk menangkap *variance* maksimal. Kita mengonfigurasi PCA agar mempertahankan 95% *variance* data asli sebelum hasilnya diteruskan ke *classifier* sederhana. Guna menegakkan *praktik pipeline yang benar*, kita menggabungkan `StandardScaler`, PCA, dan model *classifier* ke dalam satu *pipeline* utuh. Langkah ini memastikan parameter standardisasi dan proyeksi hanya dihitung dari set pelatihan, mencegah risiko *leakage* ke set pengujian.

Sebagai pembanding, kita melatih *autoencoder* dengan ukuran lapisan *bottleneck* yang sama dengan komponen utama PCA. Berbeda dengan PCA yang memaksimalkan *variance* global, *autoencoder* meminimalkan kesalahan rekonstruksi (*reconstruction error*). Jaringan saraf ini melatih enkoder untuk memetakan input ke *bottleneck*, lalu melatih dekoder untuk merekonstruksi input asli di sisi output. Pelatihan berjalan selama beberapa *epoch* hingga kesalahan rekonstruksi minimal. Transformasi non-linier ini memungkinkan *autoencoder* menangkap pola spasial dan manifold kompleks pada citra yang gagal dipisahkan oleh proyeksi linier PCA.

Karakteristik representasi dari kedua metode ini terlihat jelas saat kita memvisualisasikan ruang latennya ke dalam bentuk dua dimensi.
