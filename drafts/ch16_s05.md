# Studi Kasus: Kurasi Manusia atas Fitur Otomatis pada Data Relasional

Penerapan rekayasa fitur otomatis dan kurasi manusia dievaluasi menggunakan *dataset* ritel relasional untuk memprediksi probabilitas pelanggan berhenti berlangganan (*churn*). Data terstruktur ini tersebar di tabel Pelanggan, Pesanan, dan Detail Transaksi. Evaluasi dimulai dengan menetapkan model *baseline* menggunakan representasi dasar rancangan manusia, seperti usia pelanggan dan rata-rata pembelanjaan. Model *baseline* ini menjadi tolok ukur awal untuk memvalidasi kenaikan performa.

Untuk mengeksplorasi *ruang fitur* yang lebih luas, algoritma *Deep Feature Synthesis* (DFS) diterapkan pada skema relasional. Algoritma ini menelusuri hierarki antartabel dan merangkai berbagai primitif matematika, seperti penjumlahan transaksi bulanan atau rata-rata jeda pesanan.

Mesin sukses merampingkan data secara matematis, namun buta terhadap konteks operasional bisnis. Pakar domain kemudian menginspeksi daftar 50 fitur tersebut.
