# *Target Encoding* dan Risiko *Leakage*

Kardinalitas tinggi pada variabel kategori memicu pelebaran dimensi matriks secara masif jika dikonversi secara naif menggunakan metode biner seperti *one-hot encoding*. *Target encoding* (atau *mean encoding*) adalah teknik alternatif yang memampatkan berbagai label kategori menjadi satu lajur numerik kontinu. Representasi ini dibentuk dengan mengganti setiap string kategori menggunakan nilai rata-rata probabilitas target dari sekumpulan observasi yang memiliki label tersebut.

Kelompok kategori minoritas dengan sampel yang sangat sedikit rentan memunculkan rata-rata agregat yang fluktuatif (bervariansi tinggi). Implementasi operasional meredam anomali ini menggunakan metode pemulusan (*smoothing*). Arsitektur pemulusan berbasis *empirical Bayes shrinkage* memadukan probabilitas asli kelompok spesifik bersama rata-rata distribusi keseluruhan populasi umum:

$$ S_i = \lambda_i \mu_i + (1 - \lambda_i) \mu_{global} $$

Rasio penyeimbang probabilitas diatur oleh pembobot $\lambda_i$, yang diformulasikan menurut persentase sampel melalui persamaan:

$$ \lambda_i = \frac{n_i}{n_i + m} $$

Keterangan notasi:
*   $S_i$: Nilai numerik terenkode yang merepresentasikan kategori $i$.
*   $\mu_i$: Rata-rata keluaran nilai target lokal pada kelompok label $i$.
*   $\mu_{global}$: Rata-rata nilai target keseluruhan berdasarkan gabungan himpunan data latih.
*   $n_i$: Angka total observasi (frekuensi kejadian) kategori $i$.
*   $m$: Parameter penyetelan regulasi guna menarik variansi sampel yang renggang kembali menuju rata-rata absolut sistem.

Ketika frekuensi kelas tersebut padat ($n_i \gg m$), nilai batas $\lambda_i$ beranjak mengikat pada proporsi 1, sehingga sistem condong meyakini akurasi rata-rata kelompok ($\mu_i$). Jika kategori berstatus langka atau observasinya minim, dominasi $\lambda_i$ menciut dan model meregulasi komputasi akhir mendekat ke angka landasan empiris populasinya ($\mu_{global}$).

### Mekanisme Pemblokiran *Data Leakage*

Pemanfaatan kolom sasaran untuk mencetak parameter lajur prediksi membawa dampak destruktif laten dari perisitiwa *data leakage* (kebocoran informasi). Apabila kalkulasi perataan probabilitas dijalankan absolut menjaring observasi dari seluruh set data, matriks latih akan mencuri pandang jawaban akhir bagian data target (*test set*). Kontaminasi internal semacam ini mendemonstrasikan evaluasi metrik akurasi validasi awal yang fiktif sekaligus kegagalan total inferensi pada set produksi masa mendatang.

Kedisiplinan isolasi arsitektur diimplementasikan mandiri pada pustaka modern untuk merintangi masuknya kebocoran parameter model. Kerangka komputasi algoritma populer menawarkan jaminan integritas prediksi menumpang tiga taktik teknis dominan berikut:

*   **Pemisahan Himpunan Terisolasi:** Probabilitas spesifik variabel sepenuhnya diukur dari rentang matriks observasi data latih tanpa interpolasi baris asing. Peta target statis ini dibekukan secara logis oleh kelas *encoder*. Begitu tahapan model dihadapkan pada nilai asing sewaktu masa intervensi data inferensi, prosedur transformasinya memanggil titik tumpuan darurat cadangannya memakai target $\mu_{global}$.
*   **Validasi Silang Internal (*Cross-fitting*):** Implementasi transformer sejenis `TargetEncoder` pada pustaka scikit-learn mendelegasikan pemecahan tumpukan himpunan pelatih menembus beberapa sub-lipatan blok bergantian (*folds*). Rata-rata sasaran baris pada satu lipatan absolut diturunkan mengecualikan pengelompokannya sendiri, dan menghimpun basis angka memanen lipatan tumpukan sisanya.
*   **Statistik Target Terurut (*Ordered Target Statistics*):** Algoritma pohon keputusan mahir (CatBoost) menyabotase rantai kebocoran mengusung prinsip matematika barisan sekuensial. Ruang tabel dieksekusi menurut serangkaian permutasi pengurutan acak. Estimasi bobot terhitung menumpang jejak angka probabilitas terbatas menghitung deretan rekam jejak baris teridentifikasi yang hadir lebih dahulu di atas target pengisian bersangkutan. Mekanika komputasi asimetris satu jalur ini melenyapkan sirkulasi siklus ganda tanpa mengharuskan pertukaran pengulangan kepingan *fold*.
