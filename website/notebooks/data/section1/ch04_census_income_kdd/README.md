# Ch04 Census-Income KDD Snapshot

Source: UCI Machine Learning Repository, Census-Income (KDD).

Source URL: https://archive.ics.uci.edu/dataset/117/census+income+kdd

DOI: https://doi.org/10.24432/C5N30T

License: Creative Commons Attribution 4.0 International (CC BY 4.0).

Snapshot date: 2026-07-09.

Files:

- `census_income_kdd.zip` - original UCI download.
- `census.tar.gz` - raw tarball from the UCI zip.
- `census-income.data` - raw UCI training file.
- `census-income.test` - raw UCI test file.
- `census-income.names` - source metadata and column descriptions.
- `census_income_train.parquet` - parsed UCI training file.
- `census_income_test.parquet` - parsed UCI test file.
- `census_income_combined.parquet` - parsed combined copy used by the notebook.
- `census_income_preview_1000.csv` - small preview for quick inspection.
- `verified_stats.json` - stats recomputed from the parsed snapshot.
