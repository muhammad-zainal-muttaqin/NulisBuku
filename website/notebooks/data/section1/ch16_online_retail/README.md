# Ch16 Online Retail Automated Feature Audit

This folder reuses the Bab 6 Online Retail relational snapshot from
`data/section1/ch06_online_retail/`. It stores the Bab 16 automation audit:
baseline features, automatically generated candidate features, fixed-ruler
delta-CV scores, holdout scores, and human curation decisions.

No new source dataset is downloaded for Bab 16. The source license and
attribution are inherited from the shared Online Retail fact sheet.
